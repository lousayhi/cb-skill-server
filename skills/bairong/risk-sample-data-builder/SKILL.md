# 全局最高强制约束，不可违背
1. 覆盖7大类基础样本，可按需组合生成，支持万/十万/百万级批量构造，自动分片避免超长SQL；
   好人样本、恶意坏人样本、团伙批量样本、灰产中介样本、临界模糊样本、敏感人群样本、脏无效测试数据
2. 内置6类欺诈特征自动植入样本，可自由开关：代办中介、设备农场模拟器、虚假工作/收入、多头高频申请、冒用身份、循环养号套现；
3. 支持自定义人群分层维度：年龄区间、地域、收入档位、职业、线上/线下渠道；
4. 两种输出格式自由选择：CSV数据集（可直接导入标注工具）、MySQL批量插入SQL（批量入库测试）；
5. 每条样本自动附加完整标签字段：风险分层主标签、欺诈子类型、人群分层标签、数据质量标签；
6. 可配置各类样本生成占比，实现分层均衡数据集，适配模型正负样本均衡需求；
7. 支持自定义特殊场景：集中批量进件、异地跨城申请、短期多头征信、多账号同设备关联；
8. 输出附带中文说明、分层数量统计、入库/导入操作指引，全程中文注释；
9. 禁止闲聊、无关代码、多余表格，仅输出样本数据+分片脚本+统计汇总。

# 七大类样本内置生成规则
## 1. 好人正常样本
特征：单人单设备、资料真实统一、无多头、无逾期、稳定收入、本地长期居住、征信干净、无中介联系人
标签：好人样本
## 2. 恶意坏人样本
特征：历史多次逾期、长期失联、证件与人脸不符、虚假单位、套现循环借贷、冒用他人身份
标签：坏人样本，可附加：虚假资料/冒用身份/养号套现
## 3. 团伙批量样本
特征：同一设备ID/IP/紧急联系人批量多账号短时间集中申请，多申请人共用银行卡、手机号段高度集中
标签：团伙样本，可附加：设备农场/代办中介
## 4. 灰产中介样本
特征：短期多头借贷、小额高频循环、中介统一联系人、收入虚高、无稳定工作、异地跨区域申请
标签：灰产样本，可附加：代办/多头借贷
## 5. 临界模糊样本
特征：少量多头、短期轻微逾期、收入偏低、资料轻微瑕疵，无明确欺诈证据，风险模糊无法直接判定好坏
标签：临界样本
## 6. 敏感人群样本
特征：学生、低收入无固定职业、纯白户无征信、异地跨省申请、高龄/低龄区间
标签：敏感人群
## 7. 脏数据样本
特征：关键字段缺失、重复进件、乱码、测试账号、时间逻辑冲突、格式非法、空值填充
标签：脏数据

# 六大欺诈特征植入逻辑
1. 代办中介：统一中介紧急联系人、多申请人预留同一手机号、线下渠道集中提交
2. 设备农场：模拟器标识、单设备批量账号、短时间多IP切换、虚拟定位
3. 虚假资料：不存在工作单位、虚高月收入、虚假居住地址、证件校验不匹配
4. 多头借贷：30天数十次跨平台申请、征信查询记录超标
5. 冒用身份：实名与手机号持有人不一致、人脸核验失败记录
6. 养号套现：短期频繁放款结清、还款日集中批量还款、小额循环借贷

# 输出格式规范
## 格式1：CSV数据集
包含全部业务字段+标签字段，首行表头UTF-8编码，逗号分隔，分层样本分段区分，可直接导入标注工具、mitmproxy、压测参数池
## 格式2：MySQL批量插入SQL
可选输出建表语句，自动分片每批5000条，字段类型匹配样本数据，附带注释区分样本类型

# 完整执行流程
Step1：解析输入需求：样本类型配比、生成总量、欺诈特征开关、人群分层约束、输出格式
Step2：按配比批量生成对应特征原始业务流水，植入匹配欺诈行为、人群属性
Step3：自动为每条样本附加全套标签字段（分层标签+欺诈标签+人群标签）
Step4：千万级数据自动分片拆分，防止单段文本溢出
Step5：生成CSV文件或分片INSERT SQL脚本
Step6：输出分层样本数量统计、导入/入库执行指引

# 禁止行为清单
1. 禁止缺失7类基础样本、6类欺诈特征任意一类生成逻辑；
2. 禁止批量大数据不分片，输出超长单条SQL；
3. 禁止样本无配套标签字段，仅输出原始流水；
4. 禁止不支持自定义各类样本生成比例；
5. 禁止英文为主描述，所有注释、统计说明统一中文。
