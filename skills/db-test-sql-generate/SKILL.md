# 全局强制约束（最高优先级，不可违背）
1. 输入支持单表/多表建表语句、字段类型、长度约束、主键、唯一索引、普通索引、外键、非空、默认值；
2. 输出全程中文描述，SQL语句注释使用中文，不输出英文模板说明；
3. 必须完整产出5大类SQL，缺一不可：
   ① 基础增删改查CRUD标准测试SQL
   ② 脏数据构造SQL（超长字符、超限数值、空值、特殊字符、重复唯一值）
   ③ 分页、联表、分组统计边界查询SQL
   ④ 索引有效性校验、慢查询验证SQL
   ⑤ 事务、主从、上下游数据一致性校验脚本
4. 自动根据字段约束生成对应边界测试数据：数值最大/最小值、字符串长度临界、日期边界、枚举非法值；
5. 唯一键、主键自动生成冲突插入测试SQL；非空字段生成空值插入报错SQL；
6. 分页自动覆盖page=0、page负数、size超限、超大偏移量limit边界场景；
7. 索引校验包含：无索引全表扫描对比、索引命中验证、联合索引最左匹配测试；
8. 一致性校验覆盖：事务回滚数据校验、主从数据条数对比、关联表外键完整性校验；
9. 输出结构分层清晰，每段SQL附带中文测试目的说明，禁止纯代码无解释；
10. 仅输出中文说明+SQL代码，不输出表格、YAML、多余英文话术、无关闲聊。

# 五大模块生成规则
## 模块1：基础CRUD测试SQL
1. 正常新增：填充全部合法字段插入
2. 部分字段新增：仅必填字段插入
3. 正常单条更新：修改单个/多个普通字段
4. 唯一字段更新冲突场景SQL
5. 按主键精准删除、批量条件删除
6. 单条查询、全表分页查询、条件精准查询

## 模块2：脏数据构造SQL
自动生成各类非法测试数据插入语句：
1. 字符串超过最大长度
2. 数值超出字段存储范围
3. 非空字段传入NULL/空字符串
4. 唯一索引重复值插入
5. 特殊符号、emoji、换行符文本
6. 超大文本、超长数字字符串
7. 非法日期、时间边界值

## 模块3：分页&联表边界查询SQL
1. 分页边界：limit 0、limit 负数、超大offset、size超过最大限制
2. 多表inner/left/right联表查询，关联字段不存在场景
3. group by、count、sum统计边界（空表、仅1条数据、海量数据）
4. order by 索引字段、无索引字段对比查询
5. in 传入空数组、超大批量ID查询

## 模块4：索引校验SQL
1. 查看表全部索引结构语句
2. explain 验证普通索引、联合索引命中情况
3. 最左匹配失效测试SQL
4. 无索引全表扫描对比慢查询
5. 唯一索引、主键索引查询性能对比
6. 索引字段隐式转换失效场景验证

## 模块5：数据一致性校验脚本
1. 事务提交/回滚后数据校验SQL
2. 主从库数据条数、关键字段对比校验语句
3. 外键关联表完整性校验（孤儿数据查询）
4. 批量插入后总数校验、分页汇总数据一致性
5. 重复提交幂等数据校验SQL

# 完整执行流程
Step1：解析输入建表语句，提取字段名、数据类型、长度、约束、主键、索引、外键
Step2：分类生成CRUD基础语句，匹配字段合法参数
Step3：基于字段约束生成各类脏数据插入SQL
Step4：构造分页、联表、统计类边界查询语句
Step5：生成explain索引校验、慢查询验证语句
Step6：输出事务/主从/关联表一致性校验脚本
Step7：分层输出，每一段附带中文测试用途说明

# 禁止行为清单
1. 禁止输出英文描述，所有说明、注释统一中文；
2. 禁止缺失五大类任意一类SQL脚本；
3. 禁止忽略字段约束，不生成对应边界脏数据；
4. 禁止无索引校验、无一致性校验相关语句；
5. 禁止SQL无中文注释、无测试场景说明；
6. 禁止硬编码固定测试数据，根据表结构动态适配字段。
